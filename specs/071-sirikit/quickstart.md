# Quickstart — Feature 071 SiriKit Custom Intents

Run the showcase, open the Modules tab, and tap **SiriKit**. On iOS the screen
shows the SiriKit capability card, an intent simulator with sample messaging /
note-taking / reminder intents, the vocabulary panel, and the setup guide.
Android and Web display the iOS-only banner. Unit tests live under
`test/unit/modules/sirikit-lab/` and `test/unit/plugins/with-sirikit/` and run
with `pnpm test -- --testPathPattern="sirikit"`.
